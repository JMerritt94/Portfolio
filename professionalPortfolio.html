Project 2: Resort Me

For our second project I worked in a team of four that put together an Express app that can search for ski resorts from our MYSQL database of resorts based on name and state. We used firebase authentification to have users login and make comments on the various resorts. I also gained experience using Heroku and JawsDB for our database. I worked with the Google maps API and the sequelize search functions for the project and learned a great deal about how to integrate NPMS into a full stack app. For the front end we used Font Awesome, Bootstrap, and just basic CSS.

Burgertime app

This app uses Express, MYSQL, ES6 and Handlebars to create a dynamically updating web page that allows users to consume burgers from a menu, add new burgers to consume, and revive all of the burgers that have been eaten. The front end was done through Express Handlebars which allows me to dynamically generate front end content for each item in the MYSQL database. Learning how to generate front end content through handlebars was extremely valuable, since a back end that can update a front end can be extremely useful for building apps.
